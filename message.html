ME2U (Transfert de crédits)

Description

Le transfert de crédit de communication est un service à valeur ajoutée de MTN qui permet aux clients Pay as you Go de transférer du crédit de communication vers un autre numéro Pay as you Go en utilisant leur numéro. Ce service vous permet d'avoir la liberté de partager votre crédit de communication avec vos amis et vos proches.

Comment utiliser ce service ?

1ère Etape :

Activation du code de transfert Pour utiliser ce service, l'abonné devra activer son code de transfert en composant : *155*0000*code de transfert# puis valider (OK ou Appel) Ce code est indispensable pour pouvoir effectuer des transferts (le code de transfert est un code de 4 chiffres différent de 0000).

2ème Etape :

Transfert des crédits de consommation L'abonné MTN Pay as you Go peut effectuer des transferts de crédit dont le montant est compris entre 200F et 50.000FCFA. Pour transférer du crédit, il suffit de composer *155*Montant à transférer*Numéro du réceptionnaire*code de transfert# Ex : *155*500*97880088*1234#

Tarif

Chaque transfert vous coûtera 20FCFA quelque soit le montant transféré compris entre 200FCFA et 50.000FCFA.

N.B. :

Pour changer son code de transfert, composer : *155*code de transfert actuel*nouveau code# Après trois tentatives erronées, le code de transfert est bloqué. Pour obtenir un nouveau Code de Transfert, l'abonné devra se rendre dans une agence MTN ou appeler le service clientèle.
